# Taller01_Stakeholders_Marquez_Lopez_Pablo
## Investigación sobre los *stakeholders* y los temas materiales
El triángulo **ASG** son unos criterios de sosteibilidad poara un desarrollo de software sostenible. Las siglas **ASG** significan Ambiental, Social y Gobernanza.
Los 3 temas materiales más importantes a tratar son:
1. La Eficiencia de lo Invisible (Ambiental)
2.  La Tecnología como Herramienta de Inclusión (Social)
3. La Ética detrás de la Gestión (Gobernanza)
